# Project3_DOM
Project 3: Dynamics of Machinery 
### **Context** 

The locomotion is the study of the movement in animals. Based on this, we want to create and simulate the movement of a Red Kangaroo.

For this, we follow the next steps:
 - Understand the kinematic motion of the animal. For this, we look for literature or examples of the motion.
 - Analyse restrictions: Simulate a scale version to know the dynamics of the problem.
 - Create a basic mechanism based on the previous analysis to get a first approximation of the final robot. For this, we use MuJoco.
